# Arduino-LCD-clock
Simple clock made with Arduino and an LCD display.

## Components
| Component | Quantity |
| ----------|----------|
| Arduino UNO | 1 |
| LCD 16x2 | 1 |
| 1 kΩ potentiometer | 1 |
| 200 Ω resistor | 1 |
| Button | 2 |
| 1 kΩ resistor | 2 |
